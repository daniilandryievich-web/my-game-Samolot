# Deweloper Interfejsu i UX (Instrukcja)

**Twoje zadanie:**
Skonstruowanie nowoczesnej i interaktywnej nakładki 2D (Overlay) na renderującą się w tle grę 3D.

**Kroki do wykonania:**
1. **Menu Główne:** Ekran blokujący start gry posiadający główny przycisk uruchomieniowy. Ekran powinien estetycznie zasłaniać lub rozmywać scenę 3D znajdującą się w tle.
2. **HUD (Heads Up Display):** System działający w toku rozgrywki wyświetlający najważniejsze zmienne: aktualny wynik i powiadomienia opcjonalne.
3. **Ekran "Game Over":** Ekran końcowy pojawiający się automatycznie na skutek zdarzenia porażki w 3D. Umożliwia ponowne spięcie i restart całej gry.
4. Elementy DOM używane w 2D muszą działać znakomicie i na telefonie, i w przeglądarce PC (używaj grid/flex oraz spójnego minimalizmu CSS - unikaj domyślnego brzydkiego formatowania przeglądarki).

**Zakończenie:** Kod działa we współpracy z logiką, a interfejs w 100% ukrywa lub zamyka poprawnie mechaniki gry (nie gramy w menu głównym itp.).
