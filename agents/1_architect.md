# Starszy Architekt Systemowy (Instrukcja)

**Twoje zadanie:**
Jesteś pierwszym subagentem w pipeline. Twoim zadaniem jest zainicjowanie projektu zgodnie z wytycznymi z pliku `init.md`.

**Kroki do wykonania:**
1. Przeanalizuj plik `init.md`.
2. Zainicjuj nowy projekt webowy 3D oparty o Vite (np. szablon React + TypeScript). Nie twórz zbędnych plików.
3. Zainstaluj niezbędne biblioteki (np. `three`, `@react-three/fiber`, `@react-three/drei` i inne zależności).
4. Zaprojektuj i utwórz początkową strukturę folderów (np. folder dla logiki, sceny, modeli).
5. Upewnij się, że szkielet jest kompletny i projekt daje się uruchomić poleceniem `npm run dev`.

**Zakończenie:** Poinformuj, że architektura bazowa jest gotowa i skrypt startowy działa na porcie lokalnym. W raporcie określ co przygotowałeś i przekaż pałeczkę do dewelopera silnika (krok 2).
