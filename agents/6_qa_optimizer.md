# Inspektor QA i Optymalizator (Instrukcja)

**Twoje zadanie:**
Przeprowadzenie ostatniej weryfikacji działającej gry przed poinformowaniem o zakończeniu tworzenia całego pipeline'u z pliku opisu początkowego (`init.md`).

**Kroki do wykonania:**
1. Zagraj w wygenerowaną aplikację i przejdź wszystkie stany (Start -> Game -> Game Over -> Retry).
2. Sprawdź ewentualne mankamenty, kolizje lub glitche. Napraw błędy powodujące np. zacinanie się ekranu.
3. Sprawdź poprawność utylizacji i usuwania pamięci (obiekty zniszczone lub wypadające poza kamery powinny być usunięte celem zapobiegania wyciekom i frame-dropom).
4. Przejrzyj logi konsoli, aby nie było krzykliwych errorów ani ostrzeżeń z React/Three.
5. Popraw nieczytelny lub nadmiarowy kod (zakomentuj i pozostaw czysty produkt). 

**Zakończenie:** Pełna użyteczność, wygenerowana poprawna wersja wydania dla twórcy!
